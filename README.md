# 🤖 Natural Language Processing (NLP) Master Roadmap

Welcome to my NLP repository! This project is a structured journey through the world of computational linguistics, following the path laid out by the [NLP Roadmap](https://nlproadmap.netlify.app/). 

This repo serves as a central hub for my notes, code implementations, and project milestones as I progress from basic text processing to State-of-the-Art (SOTA) Large Language Models.

---

## 🗺️ Roadmap Progress

### 🏗️ Phase 1: Text Preprocessing & Basics
- [ ] **Tokenization & Regex:** Splitting text and pattern matching.
- [ ] **Stemming & Lemmatization:** Reducing words to root forms.
- [ ] **Stop Words & POS Tagging:** Filtering noise and identifying parts of speech.
- [ ] **Named Entity Recognition (NER):** Identifying real-world objects (Names, dates, etc.).

### 📊 Phase 2: Feature Extraction (Vectorization)
- [ ] **Bag of Words (BoW):** Simple frequency counting.
- [ ] **TF-IDF:** Term Frequency-Inverse Document Frequency.
- [ ] **Word Embeddings:** Word2Vec, GloVe, and FastText.

### 🧠 Phase 3: Deep Learning for NLP
- [ ] **RNNs & LSTMs:** Handling sequential data and long-term dependencies.
- [ ] **GRUs:** Gated Recurrent Units for efficient sequence modeling.
- [ ] **Encoder-Decoder Architectures:** The foundation for machine translation.

### ⚡ Phase 4: The Transformer Era
- [ ] **Attention Mechanism:** Learning where to "focus" in a sentence.
- [ ] **BERT:** Bidirectional Encoder Representations from Transformers.
- [ ] **GPT Models:** Generative Pre-trained Transformers.

### 🚀 Phase 5: LLMs & Modern Applications
- [ ] **Prompt Engineering:** Optimizing inputs for Generative AI.
- [ ] **RAG (Retrieval-Augmented Generation):** Connecting LLMs to external data.
- [ ] **Fine-tuning:** Adapting pre-trained models to specific domains.
